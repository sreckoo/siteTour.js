Install files

Installing the sitetour jQuery plugin fairly simple. You just need to add the following code to the <head> of your web page: 

<link rel="stylesheet" href="siteTour.css" type="text/css" />
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js" type="text/javascript"></script>
<script src="jquery.sitetour-0.4.js" type="text/javascript"></script>

Add markup

The HTML markup for the siteTour is also fairly simple. You simply need to add two basic attributes step-no and data-text to the elements that you wish to hightlight during the tour. step-no attribute must contain number value starting with 1 and then you increase it for the next step, depending on your sequence. Optionally you can add data-position attribute to specify left, right, top or bottom position of the text box appearing beside the highlighted element: 

<div step-no="1" data-text="First Step">
    <h3>First step</h3>
        Powder pudding donut apple 
        pie powder chocolate.
</div>
<div step-no="2" data-position="right" data-text="<h3>Second step</h3>">
    <h3>Second step</h3>
        Gummies ice cream chocolate cake cheesecake 
        cotton candy fruitcake cheesecake cotton candy. 
</div>

Hook up the siteTour

Finally you need to hook up the siteTour with the "click" element by adding the following code after the three links we included in the <head>. $("#trigger") below is the jQuery selector on element to be clicked:

<script type="text/javascript">
        $(function () {
            $("#trigger").siteTour();
        });
</script>

Customize the settings

$('#slider').siteTour({
    scrollPage: false,              // Scroll page in order to focus on the selected element
    nextButton: "",                 // Text for the next button
    previousButton: "",             // Text for the previous button
    closeButton: "",                // Text for the close button
    doneButton: "",                 // Text for the close button
    complete: function(){}          // Triggers when function completes
});
